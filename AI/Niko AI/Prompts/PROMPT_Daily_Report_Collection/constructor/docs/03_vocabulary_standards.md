### AI & LLM Terms

#### Prompting Techniques
- **Prompting** - The process of instructing AI models
  - Task - What the AI should do
  - Context - Background information
  - References - Examples or guidelines
  - Evaluate - Success criteria
  - Iterate - Refinement process
- **Chain of Thought** - Breaking down reasoning into steps
- **Tree of Thought** - Exploring multiple reasoning paths
- **Prompt Chaining** - Linking multiple prompts together
- **Meta Prompting** - Prompts that generate prompts

#### AI Concepts
- **Persona** - Defined character/role for AI to adopt
- **Multimodal prompting** - Using text, images, and other formats
- **Hallucinations** - AI generating false or nonsensical information
- **Biases** - Systematic errors or prejudices in AI outputs
- **Human-in-the-loop** - Workflow requiring human oversight
- **Agent Sim** - AI agent simulation
- **Agent X** - Specific agent implementation

### General Business Terms

#### Task Management
- **Task Template** - Structured task format (Action + Object + Context)

## Action + Object Framework

### Standard Actions

Actions have three forms:
1. **Action Form** - Base verb (e.g., "access")
2. **Process Form** - Present continuous (e.g., "accessing")
3. **Result Form** - Past participle (e.g., "accessed")

Common actions include:
- access, analyze, archive, audit, adjust
- back up, build, check, create, debug
- deploy, design, develop, document, edit
- implement, integrate, maintain, monitor, optimize
- review, test, update, verify

## Integration Guidelines

### In Daily Report Prompts

1. **Reference vocabulary section** in base context
2. **Apply consistent terminology** across all reports
3. **Use Action + Object format** for all tasks
4. **Standardize status and priority** levels
5. **Apply RACI matrix** where accountability is discussed
6. **Recognize and flag templates** (Task, Step, Checklist)
7. **Use profession recognition** to accurately identify roles
8. **Handle multi-language content** appropriately

### Quality Checks

- Verify all terms match standardized vocabulary
- Ensure Action + Object format for tasks
- Check that status/priority use standard values
- Confirm RACI assignments when accountability discussed
- Validate profession titles against 636+ variations
- Check multi-language terms are properly translated/transliterated
