## üßπ Clean Installation (Recommended)

For a clean installation and to work only with shared structured files, follow these steps:

### Step 1: Uninstall DropBox

**For Windows:**
1. Press **Windows key** or click **Start menu**
2. Type **"Settings"** and open Settings app
3. Click on **"Apps"** (or **"Apps & features"**)
4. In search box, type **"Dropbox"**
5. Find **"Dropbox"** application in the list
6. Click on it, then click **"Uninstall"**
7. Follow uninstall wizard prompts

**Alternative Method (Control Panel):**
1. Press **Windows key + R**
2. Type **"control"** and press Enter
3. Go to **"Programs"** ‚Üí **"Uninstall a program"**
4. Find Dropbox, right-click, select **"Uninstall"**

**For Mac:**
1. Open **Finder**
2. Go to **Applications**
3. Find **Dropbox** and drag it to **Trash**
4. Empty Trash

### Step 2: Clean Up Remaining Files (Optional but Recommended)

1. Navigate to your Dropbox folder location:
   - **Windows:** Usually `C:\Users\[YourName]\Dropbox`
   - **Mac:** Usually `/Users/[YourName]/Dropbox`
2. Delete any remaining files/folders if they exist
3. This ensures clean reinstallation

### Step 3: Download Fresh DropBox Installer

1. Open your web browser
2. Visit one of these official DropBox download pages:
   - **Primary:** https://www.dropbox.com/install
   - **Alternative:** https://www.dropbox.com/downloading
3. Click the **"Download"** button
4. **Important:** Download the **Desktop app**, NOT the web version
5. Skip any prompts for web version
6. Save installer to Downloads folder or Desktop

### Step 4: Install DropBox

1. Navigate to Downloads folder
2. Double-click the Dropbox installer file
3. Follow installation wizard:
   - Click **"Next"** through initial screens
   - Accept license agreement if prompted
   - Choose installation location (default is fine)
   - Click **"Install"**
4. Wait for installation to complete
5. Click **"Finish"** when done

### Step 5: Login with New Account

1. Dropbox Desktop should launch automatically after installation
2. If not, search for **"Dropbox"** in Start menu (Windows) or Applications (Mac) and open it
3. Login window will appear
4. **Enter your new account credentials:**
   - **Designers:** `dd@rh-s.com`
   - **Video Editors:** `contact@fv-e.com`
5. Complete authentication if 2FA is enabled
6. Wait for Dropbox to initialize
