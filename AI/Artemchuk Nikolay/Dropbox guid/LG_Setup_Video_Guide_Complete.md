## Overview

This comprehensive video guide provides detailed, scene-by-scene instructions for setting up all essential tools for LG team members. The guide features **Axie the Axolotl** as your learning companion throughout every step, with **Mary the Wolf** providing expert guidance. Each scene is designed to be clear, visual, and actionable, enabling employees to complete the entire setup process independently.

**Tools Covered:**
- Dropbox Desktop (clean installation, LG account, selective sync)
- VS Code or Cursor IDE (installation and configuration)
- Claude Code Extension (installation and authentication)
- Git Bash (installation and verification)

**Total Setup Time:** 30-45 minutes (plus this video guide duration)
