# Dropbox Setup Guide - Project Files

This folder contains all files related to the LG Team Setup Instructions explainer video project.

## Contents

### 1. LG_Setup_Instructions_Complete_Guide.md
Complete step-by-step setup guide for LG team members. Includes:
- Dropbox setup (uninstall, fresh install, selective sync)
- IDE setup (VS Code or Cursor)
- Claude Code Extension setup
- Git Bash installation
- Verification and testing
- Troubleshooting guide

### 2. LG_Setup_Video_Scenario.md
Comprehensive video scenario document for creating explainer videos about the setup process. Includes:
- Multiple video format options (short, medium, long form)
- 7 detailed scenarios covering all setup steps
- Visual design guidelines using Mary the Wolf mascot
- Platform-specific adaptations (Instagram, TikTok, YouTube, LinkedIn)
- Production notes and checklists
- Follows Remote Helpers (RHS) design guidelines
