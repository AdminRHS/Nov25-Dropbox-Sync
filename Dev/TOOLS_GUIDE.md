## Tool Categories

### Primary Tools (Used Daily)

#### Claude
- **Vendor:** Anthropic
- **Purpose:** Advanced AI assistant for analysis, automation, and MCP connector integration
- **Category:** AI Tools / LLM Services
- **Skill Level:** Beginner to Advanced (depending on use case)
- **Cost:** API-based (usage pricing), Subscription for Pro features
- **Documentation:** https://claude.ai

#### Claude Desktop App
- **Vendor:** Anthropic
- **Purpose:** Local desktop application for Claude AI with MCP connector support
- **Category:** AI Tools / Application
- **Skill Level:** Intermediate
- **Cost:** Free application, requires Claude API subscription
- **Documentation:** https://claude.ai/download

#### Claude Projects
- **Vendor:** Anthropic
- **Purpose:** Organize AI workflows with custom knowledge, instructions, and team collaboration
- **Category:** AI Tools / Project Management
- **Skill Level:** Beginner to Intermediate
- **Cost:** Included with Claude Pro subscription ($20/month) and Team plans
- **Documentation:** https://claude.ai/projects
