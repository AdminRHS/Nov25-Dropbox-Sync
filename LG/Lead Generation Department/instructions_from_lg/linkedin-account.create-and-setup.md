# **LinkedIn Profile Visibility Settings Guide**

## **Basic Visibility Settings**

* **Step 1**: Access Privacy Settings  
  * Click "Me" icon at top of homepage  
  * Select "Settings & Privacy" from dropdown  
  * Navigate to "Visibility" section  
  * Click "Edit your public profile"  
* **Step 2**: Profile Visibility Mode  
  * Toggle "Your profile's public visibility" to ON  
  * Set visibility to "Public" for maximum professional exposure  
  * Ensure "Make my public profile visible to everyone" is enabled

## **Individual Section Visibility**

* **Step 3**: Customize Section Visibility Set the following sections to "Public":  
  * Profile photo  
  * Background image  
  * Headline  
  * Current work experience  
  * Education  
  * Location  
  * Industry  
  * Skills & endorsements  
  * Contact info (professional only)  
* **Step 4**: Hide Personal Information Set to "Private" or "Connections Only":  
  * Birthday  
  * Phone number  
  * Email address (unless business email)  
  * Connection list (optional)  
  * Salary information

## **Profile Viewing Options**

* **Step 5**: Set Profile Viewing Settings  
  * Go to "Profile viewing options"  
  * Select "Your name and headline" (not private mode)  
  * This allows others to see when you view their profile  
  * Helps in networking and lead generation

## **Story Visibility**

* **Step 6**: Configure Story Settings  
  * Navigate to "Visibility of your LinkedIn Story"  
  * Set to "Public" for maximum reach  
  * Enable "Allow reshares" for better content distribution

## **Search Engine Privacy**

* **Step 7**: Search Engine Settings  
  * Find "Profile visibility off LinkedIn"  
  * Enable "Yes, show my profile to search engines"  
  * This allows your profile to appear in Google searches  
  * Increases visibility to potential clients/employers

## **Activity Feed Settings**

* **Step 8**: Manage Activity Visibility  
  * Go to "Activity broadcast"  
  * Enable sharing profile changes  
  * Set article/post visibility to "Public"  
  * Allow endorsement visibility  
  * Enable recommendation visibility

## **Network Visibility**

* **Step 9**: Connection Settings  
  * Set "Who can see your connections" based on strategy:  
    * Set to "Public" if building open network  
    * Set to "Connections only" if working with sensitive clients  
  * Enable "Allow connections to see your email"  
  * Enable "Allow LinkedIn to show your profile on others' 'People also viewed'"

## **Profile Status**

* **Step 10**: Active Status Settings  
  * Enable "Share active status"  
  * Allow "Show when you're on LinkedIn"  
  * This helps with real-time networking

## **Verification Steps**

* **Step 11**: Final Visibility Check  
  1. Open an incognito/private browser window  
  2. Search for your profile on LinkedIn  
  3. Verify all intended sections are visible  
  4. Check if your profile appears in Google search  
  5. Review profile from a connection's view  
  6. Ensure all professional information is visible

# **Bans Reasons for LinkedIn Account** 

## **Registration Red Flags**

1. **VPN Usage**  
   * LinkedIn detects VPN connections  
   * Suspicious IP address changes  
   * Multiple accounts from same IP  
2. **Email Issues**  
   * Using temporary email services  
   * Recently created email accounts  
   * Multiple accounts with similar emails  
3. **Profile Creation Speed**  
   * Filling out profile too quickly  
   * Adding many connections instantly  
   * Automated profile completion  
4. **Suspicious Patterns**  
   * Multiple logins from different locations  
   * Rapid connection requests  
   * Using scripts or automation tools

# **Prevention Strategies**

## **Account Setup**

1. **Email Preparation**  
   * Gmail or business email preferred  
   * Avoid free temporary email services  
2. **Connection Method**  
   * Use stable home/office internet  
   * Avoid public WiFi for initial setup  
   * Don't use VPN during registration  
3. **Profile Creation**  
   * Take 1-2 days to complete profile  
   * Add information gradually  
   * Use manual input, avoid copy-paste  
   * Wait 24-48 hours before adding connections

## **Natural Growth Pattern**

1. **First Week**  
   * Complete basic profile sections  
   * Add profile picture after 24 hours  
   * Max 15-20 connection requests per day  
   * Engage with content naturally  
2. **Second Week**  
   * Complete remaining profile sections  
   * Join 2-3 relevant groups  
   * Follow companies gradually  
   * Like/comment on posts occasionally  
3. **Long-term Strategy**  
   * Build network slowly (20-25 connections/day)  
   * Mix activities (posts, comments, likes)  
   * Maintain consistent login patterns  
   * Use same device/location regularly

## **Security Measures**

1. **Login Habits**  
   * Use same browser consistently  
   * Enable two-factor authentication (with the registered earlier email)   
   * Keep login location stable  
   * Avoid suspicious activity plugins  
2. **Profile Authenticity**  
   * Use real personal information  
   * Upload genuine profile photo  
   * Provide accurate work history  
   * Include verifiable details  
3. **Activity Guidelines**  
   * No automation tools  
   * No mass connection requests  
   * No scraping or data harvesting  
   * Respect LinkedIn's daily limits

## **Recovery Options**

If banned:

1. Contact LinkedIn Support  
2. Provide identity verification  
3. Explain legitimate business purpose  
4. Submit appeal with documentation  
5. Be patient during review process
