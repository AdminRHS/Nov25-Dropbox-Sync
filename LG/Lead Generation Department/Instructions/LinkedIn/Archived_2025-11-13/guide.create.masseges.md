**1\. Fill in the LG Table after a lead accepts your connection**

* Open the shared **LG table** with your LG account.

* Add the following information:

  * **Date** (today’s date)

  * **LG Manager** (your name)

  * **Company name**

  * **Company website** (from LinkedIn → About → Website)

  * **CRM link** (card of this company)

  * **Lead name**

  * **Lead link** (LinkedIn profile of the lead)

  * **Status**: choose between

    * Send message (you wrote the first message)

    * Interested / Not interested

---

**2\. Generate Deep Research**

* Use the **AI tool (e.g., Grok)**.

* **Create a new chat** in the AI tool and name it with **your own name**.

* This will be your **one and only chat** where you do **all deep research** for companies.

* Copy the **prompt** (company name, lead name, website, LinkedIn page, other links).

* Paste it into your personal chat.

* Wait for the report → copy the **whole reply**.

* Save it in **Google Drive** to your personal folder → create a new Google Doc → paste the research → rename file with the company name.

* Copy the **Google Doc link** and paste it in the LG table under “Deep Research.”

* Also copy the **AI chat link** (where you generated the research) and paste it into the table.

---

**3\. Save in Dropbox**

* After creating a Deep Research file in **Google Drive**, download it as Markdown (.md).  
  ![][image1]

* Save it into your personal **Dropbox folder**.  
  

---

**First message**: After Deep Research, generate the first message in **ChatGPT (LG account, your folder)**.

![][image2]

* Make sure to **choose “thinking” mode** before generating the message.

![][image3]

**Second, third, fourth messages**: If the lead doesn’t respond, generate follow-up messages the same way (in the same chat with the AI).

Add each follow-up status to the LG table (2nd, 3rd, etc.).

**4\. Summary of what you must log in the LG Table**

* Lead name

* Lead link

* Status (connected / message sent / interested / not interested)

* CRM link

* Company name

* Company website

* Deep Research (Google Doc link)

* AI tool used (Grok, Gemini, etc.)

* Chat link (AI chat link)

* Your name (LG manager)

* Date
