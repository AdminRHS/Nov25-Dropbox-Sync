# Onboarding & Training

This folder contains all materials for new Lead Generation Manager onboarding and training. These resources guide new hires through their first days and weeks, covering company structure, department workflows, essential vocabulary, and initial setup requirements.

## What's Inside:

### Onboarding Programs
- **full-docs-lg-onboarding.md** - Complete comprehensive onboarding documentation
- **instructions.full-lg-onboarding.md** - Full LG onboarding process guide
- **instructions.first-week.onboarding.md** - First week orientation and tasks
- **instuctions.lg-manager-onboarding.md** - LG Manager specific onboarding

### Getting Started
- **STARTER PACK-RH.md** - Essential starter pack for Remote Helpers
- **Tutorial_for_beginners_Remote_Helpers.md** - Beginner's guide for new team members
- **introduction.to-the-lg-department.md** - Department introduction and overview

### Setup & Access
- **instuctions.join-channels.md** - How to join communication channels
- **List_of_loom_Lessons.md** - Video lesson catalog and learning resources

### Knowledge Base
- **lg.vocabulary.md** - Lead generation terminology and vocabulary reference
- **common.interview-questions.lg.md** - Interview preparation questions for LG positions
- **instructions.hire-lg.md** - Hiring process and criteria for LG positions

## Who Should Use This:

- **New Lead Generation Managers** - Follow onboarding materials sequentially
- **Team Leads** - Use these guides when conducting onboarding sessions
- **HR Staff** - Reference for managing new hire orientation process
- **Managers** - Review hiring and onboarding standards

## Recommended Learning Path:

1. Start with **introduction.to-the-lg-department.md** for overview
2. Review **lg.vocabulary.md** to learn essential terminology
3. Follow **instructions.first-week.onboarding.md** for week-one tasks
4. Complete **full-docs-lg-onboarding.md** for comprehensive training
5. Watch videos from **List_of_loom_Lessons.md** for visual learning
6. Join channels using **instuctions.join-channels.md**

## Key Learning Objectives:

- Understand Lead Generation Department structure and goals
- Learn essential LG terminology and processes
- Complete all first-week setup requirements
- Access all necessary communication channels
- Build foundation for successful LG work

## Support:

If you have questions during onboarding, contact your Team Lead or reach out in the designated onboarding support channel.
