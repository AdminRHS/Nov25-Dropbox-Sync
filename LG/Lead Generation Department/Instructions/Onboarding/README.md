# Onboarding Folder - Lead Generation Department

**Complete 5-day onboarding program and comprehensive guides for new Lead Generation Managers.**

---

## A) Folder Overview

### What This Folder Contains

This folder contains the **complete onboarding curriculum** for new Lead Generation Managers at Remote Helpers. It includes a structured 5-day training program (presented in Ukrainian), comprehensive general guides covering all essential skills, reference materials, and vocabulary resources.

### Topics & Processes Covered

1. **Day-by-Day Training Program (Days 1-5)** - Interactive, hands-on onboarding sessions covering fundamentals through advanced CRM workflows
2. **Account Setup** - Gmail, Chrome profiles, LinkedIn accounts, and professional digital presence
3. **Lead Generation Fundamentals** - Understanding leads, target audiences, ideal customer profiles (ICP), and search strategies
4. **Tool Mastery** - Google Sheets, CRM systems, Chrome extensions (RE Extension, Google Scrape, SalesQL)
5. **Research & Outreach** - Deep research techniques, AI-powered message generation, personalized communication
6. **CRM Management** - Company and lead creation, data accuracy, status tracking, and pipeline hygiene
7. **Communication Channels** - Discord setup, team collaboration, and reporting workflows
8. **Call Scheduling** - Google Calendar management, event booking, and sales coordination
9. **Department Culture** - Core values, success metrics, best practices, and continuous improvement

### General Purpose

The documents in this folder enable new Lead Generation Managers to:
- Complete a structured 5-day onboarding journey from zero to productive team member
- Set up all required tools, accounts, and professional digital presence
- Master lead generation techniques, CRM workflows, and outreach strategies
- Understand department mission, responsibilities, and performance expectations
- Navigate team communication channels and collaboration tools
- Access reference guides for ongoing learning and quick problem-solving
- Build foundational knowledge of lead generation vocabulary and concepts

## I) Teaching & Facilitation Notes (For Team Leads)

### Onboarding Best Practices

**Format:**
- Interactive with cameras and mics on
- Encourage questions at any moment
- Screen sharing rotation (every participant demonstrates)
- Learn together, make mistakes together
- Friendly, supportive atmosphere

**Pacing:**
- Follow timing guidelines but be flexible
- Check comprehension before moving forward
- Use real examples from actual companies
- Celebrate small wins (first lead, first CRM entry, first automation)

**Engagement Techniques:**
- Brainstorming sessions (What is a lead? What data do we need?)
- Collaborative document editing (Google Docs/Sheets together)
- Turn-taking for demonstrations
- Peer feedback and support
- Regular Q&A breaks

**Quality Checks:**
- Review each participant's work before progressing
- Provide specific, actionable feedback
- Celebrate good work publicly
- Address mistakes constructively
- Track progress in shared spreadsheet

### Common Challenges & Solutions

| Challenge | Solution |
|-----------|----------|
| Participant struggles with Google Sheets | Slow down, demonstrate again, pair with experienced peer |
| Extension installation fails | Switch to manual install via ZIP, check Developer Mode |
| Difficulty finding decision-makers | Show 3-5 examples, emphasize "People" tab workflow |
| Generic messages (not personalized) | Demonstrate bad vs. good examples side-by-side |
| Overwhelmed by information volume | Break into smaller chunks, focus on Day 1 only first |
| Language barriers (Ukrainian content) | Provide translation support, use visual demos |
| Falling behind schedule | Adjust timing, prioritize hands-on practice over theory |
| Low engagement/energy | Take breaks, use energizers, highlight achievements |
