6. **Bonus Impact**: Multiple sections emphasize that CRM accuracy directly affects bonus calculations (leads added, calls booked, quality metrics).

### ğŸ’¡ **Best Practices Highlighted**

- **Deduplicate before adding** - Search by company name, website, AND LinkedIn URL
- **Research first** - Verify HQ location, headcount, industry, and company activity before creating cards
- **Document every transition** - Add notes explaining status changes and next steps
- **100+ connections minimum** - Only add leads with 100+ LinkedIn connections (500+ preferred)
- **Mark calls as "Happen"** - Required for bonus and pipeline metrics
- **Quarterly refresh** - Review all owned companies every 3 months for updates

### ğŸ”§ **Technical Details**

- **RE (Remote Helpers) Chrome Extension** auto-scrapes data from LinkedIn profiles and company pages
- **CRM Panel Sections**: Leads, Leads Test, My Lead Accounts, Lead Reports (LG Daily)
- **Required Screenshot Frequency**: Every 2-4 hours (desktop screenshots, not phone photos)
- **Time Tracking**: Auto-tracks while logged in, requires clock in/out

### ğŸ“Š **Reporting Requirements**

Daily reports must include:
- LG Manager name
- Date
- Country
- Created (companies)
- Sent Request (connection requests)
- Connected (accepted connections)
- Replies (positive or negative)
- AI Tools used
