# Tools Folder - Lead Generation Department

**Complete reference for all software tools, extensions, integrations, and platform access used in lead generation workflows.**

---

## A) Folder Overview

### What This Folder Contains

This folder is the **central hub for all tool-related documentation** used by the Lead Generation Department. It covers everything from AI assistants and browser extensions to VPN security, account setup procedures, and platform integrations.

### Topics & Processes Covered

1. **AI Tools** - ChatGPT, Claude, Perplexity, Gemini, and Grok for research and message generation
2. **Chrome Extensions** - Google Scrape, RE Extension, SalesQL, and bookmark organization
3. **Account Access Setup** - Gmail and LinkedIn account creation and security workflows
4. **Platform Integrations** - Sales Navigator, CRM time tracking, and tool authentication
5. **Security & Privacy** - VPN usage guidelines for lead generation activities
6. **Browser Organization** - Bookmark management and workspace hygiene

### General Purpose

The documents in this folder enable lead generation specialists to:
- Set up their complete technical workspace from scratch
- Access and properly configure all required tools and platforms
- Maintain security, compliance, and organizational standards
- Maximize efficiency through proper tool usage and browser organization
- Troubleshoot common technical issues independently

### Critical Security Reminders

- Always use VPN when accessing location-restricted content
- Keep work and personal Chrome profiles separate
- Never share CRM login credentials
- Follow LinkedIn connection limits (15-20/day week 1, 20-25/day after)
- Update extensions immediately when new versions are released
- Enable MFA where available

### Common Pitfalls to Avoid

- **Don't skip account warming** - LinkedIn flags brand-new accounts with high activity
- **Don't ignore security settings** - Disable search personalization in Google
- **Don't use free VPNs** - They often sell user data or contain malware
- **Don't delete extension folders** - Moving/deleting breaks Chrome extensions
- **Don't waste SalesQL credits** - Prioritize decision-makers only
- **Don't mix personal and work bookmarks** - Keep Chrome profiles isolated
