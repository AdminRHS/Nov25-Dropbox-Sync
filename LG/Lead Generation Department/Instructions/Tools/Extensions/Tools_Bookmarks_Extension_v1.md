### Folder Creation

**How to Create a Folder:**
1. Right-click on bookmarks bar
2. Select **"Add folder"**
3. Name it (use emojis for visual identification!)
4. Drag bookmarks into the folder

**Folder Naming Tips:**
- âœ… Use emojis: `ğŸ“‚ Lead Gen Tools` (stands out visually)
- âœ… Keep names short: `Tools` not `All Lead Generation Tools and Extensions`
- âœ… Be consistent: All folders same format
- âŒ Avoid: `New Folder (1)`, `Untitled`, vague names

---

### Drag & Drop Organization

**Reordering Bookmarks:**
- Click and hold a bookmark
- Drag it to desired position
- Release to drop
- Works for both bookmarks and folders

**Moving Between Folders:**
1. Click and hold bookmark
2. Drag over folder name (folder opens)
3. Drop bookmark inside folder
4. Folder automatically closes

**Pro Tip:** Hold `Ctrl` (Windows) or `Cmd` (Mac) while dragging to COPY instead of move

**How to Enable:**

## TROUBLESHOOTING

### Bookmarks Not Syncing

**Check:**
1. Signed in to same Google account on all devices?
2. Sync enabled? Settings â†’ Sync â†’ Bookmarks âœ…
3. Internet connection stable?
4. Try: Sign out and back in to Chrome

**Force Sync:**
- Settings â†’ Sync â†’ Manage sync â†’ Sync now

---

### Bookmarks Disappeared

**Recovery Steps:**
1. Check Bookmark Manager (`Ctrl/Cmd + Shift + O`)
2. Look in "Other bookmarks" folder (might have moved)
3. Check "Recently deleted" if available
4. Restore from exported backup (if you have one)

**Prevention:**
- Export bookmarks monthly to Google Drive
- Enable Chrome sync
- Don't delete "Bookmarks bar" folder itself

---

### Bookmark Bar Not Showing

**Fix:**
- Press `Ctrl + Shift + B` (Windows) or `Cmd + Shift + B` (Mac)
- Or: `â‹®` menu â†’ Bookmarks â†’ Show bookmarks bar

**Check:**
- Are you in correct Chrome profile?
- Is Chrome in fullscreen? (Exit fullscreen to see bar)

---

### Too Many Bookmarks

**Symptoms:**
- Bookmarks bar overflows off screen
- Hard to find anything
- Visual clutter

**Solutions:**
1. Create folders (reduce 30 bookmarks to 5 folders)
2. Remove duplicates
3. Delete unused bookmarks
4. Use nested folders (folders within folders)
5. Archive old projects to separate folder

**Rule:** If you haven't used it in 30 days, delete or archive it

---

### Favicon Not Loading

**Issue:** Bookmark shows default globe icon instead of website's logo

**Fixes:**
1. Visit the website again (Chrome refreshes icon)
2. Delete and re-add bookmark
3. Clear browser cache: Settings â†’ Privacy â†’ Clear browsing data
4. Wait 24 hours (icons sometimes load slowly)
