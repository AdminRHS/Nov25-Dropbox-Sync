## Time Zone Management:

**Critical: Always Confirm Timezones**

**Best Practices:**
- Always include timezone in all communications
- Use tools like World Time Buddy to avoid errors
- When in doubt, confirm: "That's [Time] your time in [Timezone], correct?"
- Use timezone abbreviations: PST, EST, CET, etc.

**Common Timezone Mistakes:**
- Assuming everyone is in your timezone
- Forgetting daylight saving time changes
- Using ambiguous time formats

**Prevention:**
```
Wrong: "Let's meet Tuesday at 2 PM"
Right: "Let's meet Tuesday, March 15th at 2:00 PM EST (that's 11:00 AM PST)"
```

## Rescheduling Events:

**When Lead Requests Reschedule:**

**Immediate Response:**
```
"No problem at all! I understand things come up.

What times work better for you?
[Propose 3 new time slots]
```

**Process:**
1. Confirm new time with lead
2. Update existing calendar event (don't create new)
3. Send update notification to all attendees
4. Update CRM with reschedule note
5. Confirm new time with lead via message

**If Lead Cancels Without Rescheduling:**
```
"I understand. Would you like to reschedule for a future date,
or should I follow up with you in a few weeks?"
```

## No-Show Protocol:

**If Lead Doesn't Show Up:**

**Immediate Actions (0-5 minutes after start time):**
1. Send quick message: "Hi [Name], joining now. Let me know if you're running late!"
2. Wait 10 minutes
3. Send email if still no show

**Follow-Up Message (after 10 min):**
```
"Hi [Name],

I was looking forward to our call today at [Time], but it seems we may have had a scheduling conflict.

No worries - would you like to reschedule?
I have availability [propose times].

Let me know what works!

Best,
[Your Name]
```

**CRM Update:**
- Add note: "No-show on [Date]"
- Keep status as "Event" temporarily
- If no response after 3 days â†’ "No Response"

## Event Types:

### Discovery Call (Most Common)
- Duration: 30 minutes
- Purpose: Explore fit and needs
- Attendees: Lead + Sales Rep
- Preparation: Research and initial qualification

### Technical Call
- Duration: 45-60 minutes
- Purpose: Deep technical discussion
- Attendees: Lead + Multiple team members
- Preparation: Technical questions prepared

### Follow-Up Call
- Duration: 15-30 minutes
- Purpose: Answer questions or continue discussion
- Attendees: Varies
- Preparation: Review previous call notes

## Troubleshooting:

**Issue: Lead didn't receive calendar invite**
- Verify email address spelling
- Check spam folder
- Resend invitation
- Send meeting details via message as backup

**Issue: Wrong timezone set**
- Immediately send correction message
- Update calendar event
- Apologize for confusion
- Confirm new time clearly

**Issue: Sales team not included**
- Update event immediately
- Send direct notification to sales
- Add sales@rh-s.com to all future events

**Issue: Double-booked time slot**
- Contact lead immediately
- Sincerely apologize
- Offer multiple alternative times
- Prioritize their convenience
