## 1. General Overview

### 1.1 What This Folder Contains

The **Instructions** folder is the **central knowledge repository** for the Lead Generation Department at Remote Helpers. It contains:

- **Onboarding programs** (5-day structured training for new hires)
- **Tool documentation** (Gmail, LinkedIn, Chrome extensions, VPN, CRM integrations)
- **Process guides** (lead generation methods, outreach strategies, CRM workflows)
- **Message templates** (LinkedIn outreach, email campaigns, follow-up sequences)
- **Recruitment materials** (hiring playbooks, interview questions)
- **Reference documents** (vocabulary, best practices, troubleshooting guides)

**Total Subfolders:** 8 active folders + 1 archived
**Estimated Document Count:** 50+ active instruction files
**Primary Format:** Markdown (.md) files for universal accessibility

### 2.2 Detailed Subfolder Descriptions

#### 2.2.1 Onboarding Folder

**Purpose:** Complete 5-day training program for new Lead Generation Managers

**Primary Users:** New hires (Days 1-5), Team Leads (facilitation), HR (tracking)

**Key Documents:**
- **Day 1 (onboarding_day1.md):** 2-2.5 hours
  - What is a lead, Google Sheets fundamentals, first lead capture
  - Interactive theory, brainstorming, screen sharing rotations

- **Day 2 (onboarding_day2.md):** 4 hours
  - Manual CRM company creation, RE Extension, Google Scrape automation
  - Target: Add 2-3 companies manually, 5+ with automation

- **Day 3 (onboarding_day3.md):** 3-3.5 hours
  - Deep Research methodology, New Message Templates table
  - ChatGPT message generation, Discord channel onboarding

- **Day 4 (onboarding_day4.md):** 4-5 hours
  - Job Site table mastery, processing 10 real companies end-to-end
  - Full workflow integration, independent practice

- **Day 5 (onboarding_day5.md):** Variable duration
  - Advanced CRM workflows, call booking procedures
  - Video lessons 6-9, comprehensive assessments

**GeneralGuides Subfolder:**
- **Department Overview:** Mission, responsibilities, tools, values, success metrics
- **Join Channels:** Complete Discord setup (9 text channels, 4 voice channels)
- **Manager Path:** 49-slide presentation covering entire LG manager journey
- **Beginner Tutorial:** Gmail, Chrome, LinkedIn, SalesQL step-by-step setup
- **Starter Pack:** Templates, search patterns, calendar descriptions, message templates
- **Vocabulary:** 150+ terms across 7 categories (lead generation, qualification, marketing, metrics, persona, sales, tools)

**Dependencies:**
- References Tools folder for technical setup (Gmail, LinkedIn, extensions)
- Leads into LeadGen folder for prospecting techniques
- Integrates with CRM folder for pipeline workflows
- Uses LinkedIn folder for message templates

**Success Benchmarks:**
- Week 1: Process 10+ companies, pass 80%+ on assessments
- Month 1: 50+ companies/day, 10-15% connection acceptance
- Quarter 1: 15-20% acceptance, 5+ calls scheduled weekly
