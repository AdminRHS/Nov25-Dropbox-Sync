## ðŸŽ¬ SCENARIO SELECTION TOOL

### When choosing which scenario to create, ask:

**Impact Questions:**
- How many people will use this?
- Does it solve a common problem?
- Will it save significant time?
- Is it blocking other learning?

**Feasibility Questions:**
- Do we have access to tools?
- Can we show real examples?
- How long to produce?
- How quickly will it become outdated?

**Strategic Questions:**
- Does it fill a gap?
- Does it showcase our capabilities?
- Will it attract new clients/employees?
- Is it shareable/marketable?

### Score each scenario 1-5 on each question, prioritize highest scores.
