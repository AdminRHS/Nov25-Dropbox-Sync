- **Problem:** Tool generates wrong format for platform
- **Real Example:** "Nano Banana doesn't natively support 9:16..."

  1. Manual crop after generation
  2. Use tools with format selection (D-ID, HeyGen)
  3. Resize in editing software

**Problem 2: Unnatural Lip Sync (4:00-5:30)**
- **Visual:** Mouth movements not matching audio
- **Problem:** Lips don't sync with voice
- **Causes:** Poor audio quality, complex scripts, tool limitations
- **Solutions:**
  1. Use clearer audio/voice
  2. Simplify script pacing
  3. Add pauses in script
  4. Try different voice options
  5. Regenerate with adjusted timing
- **Real Example:** From Hagen work

**Problem 3: Motion Looks Artificial (5:30-7:00)**
- **Visual:** Jerky, unnatural movements
- **Problem:** AI motion doesn't look realistic
- **Solutions:**
  1. Use better motion prompts
  2. Try Runway Gen-3 instead of cheaper tools
  3. Combine multiple tools (static + motion separately)
  4. Use subtle motion rather than dramatic
- **Real Example:** "Surprisingly got good results on first try in Runway by..."

**Problem 4: Inconsistent Character Across Shots (7:00-8:30)**
- **Visual:** Character looks different in each frame
- **Problem:** AI generates variations of character
- **Solutions:**
  1. Use reference images
  2. Same seed/settings in image generator
  3. Generate all needed angles at once
  4. Use tools with character consistency (Leonardo AI, Midjourney with character reference)

**Problem 5: Low-Quality Export (8:30-10:00)**
- **Visual:** Pixelated, compressed video
- **Problem:** Final video looks bad
- **Solutions:**
  1. Check export settings (1080p minimum)
  2. Use higher bitrate
  3. Generate in higher resolution
  4. Use pro tool versions
  5. CapCut export optimization

**Problem 6: Facial Artifacts/Glitches (10:00-11:30)**
- **Visual:** Weird face distortions
- **Problem:** AI creates unnatural facial features
- **Solutions:**
  1. Use better source images
  2. Avoid extreme angles
  3. Regenerate multiple times
  4. Use different AI model
  5. Fix in post with editing

**Problem 7: Background Issues (11:30-13:00)**
- **Visual:** Messy or irrelevant backgrounds
- **Problem:** Background distracts from character
- **Solutions:**
  1. Generate with simple backgrounds
  2. Remove background (Runway, Remove.bg)
  3. Replace in editing
  4. Use green screen feature if available

**Problem 8: Audio Quality Problems (13:00-14:30)**
- **Visual:** Muffled, robotic voice
- **Problem:** AI voice sounds fake
- **Solutions:**
  1. Use better voice models (ElevenLabs)
  2. Add background music to mask issues
  3. Record real voice and use lip sync
  4. Adjust voice settings (pitch, speed)

**Problem 9: Timing and Pacing Issues (14:30-16:00)**
- **Visual:** Video feels too fast/slow/awkward
- **Problem:** Wrong pacing for message
- **Real Example:** "Some clips needed to be extended, some accelerated, some slowed down..."
- **Solutions:**
  1. Edit script for better flow
  2. Adjust timing in CapCut
  3. Add pauses strategically
  4. Speed up/slow down sections
  5. Use transitions to smooth pacing

**Problem 10: Platform Rejection (16:00-18:00)**
- **Visual:** Error messages from social platforms
- **Problem:** Platform won't accept AI video
- **Solutions:**

  4. Check platform policies
