## Task 6: Document Portrait Format Standards


### Steps:
1. Document the dual-format portrait requirement based on Azar Imranov example
2. Create step-by-step guide for portrait creation
3. Specify technical requirements (frame size, actual PNG dimensions)
4. Explain use cases (CRM upload vs. video interview needs)
5. Add examples with before/after or annotated screenshots
6. Share documentation with design team
7. Add to department procedures/wiki


### Resources and Links:
- [Azar Imranov Portrait Example](https://docs.google.com/spreadsheets/d/1FAbgx0KGP2qezg6Y_HYbhXx-Aj9OB4j-4LZgW3euono/edit?pli=1&gid=0#gid=0&range=102:102)
- Figma portrait frames
- Portrait tracking spreadsheet


### Instructions:

**Documentation Content:**

**Portrait Format Standard: Dual-Purpose Design**

**Purpose:**
Employee portraits must serve two functions:
1. CRM upload (requires 1000x1000 display format)
2. Video interview production (requires extended composition to waist level)

**Technical Specifications:**

*Figma Frame Setup:*
- Frame dimensions: 1000x1000 pixels
- Frame displays shoulders-up composition suitable for CRM

*Actual PNG Image:*
- Image height: Significantly taller than 1000px (typically 1400-1600px)
- Image width: 1000px (matches frame width)
- Composition: Extends to mid-chest or waist level
- Hidden portion: Extra height below frame boundaries contains extended composition

**Creation Process:**

1. Photograph/create portrait with subject framed to waist or mid-chest
2. Ensure subject is properly centered with good headroom
3. Export as PNG with full waist-level composition (1000px width, ~1400-1600px height)
4. Place PNG in 1000x1000 Figma frame
5. Position image so shoulders-up view appears in frame
6. Verify hidden lower portion contains clean mid-chest/waist composition
7. Export frame view for CRM (1000x1000)
8. Maintain original PNG for videographer access

**Usage:**
- CRM System: Use 1000x1000 exported frame view
- Video Interviews: Videographers access full PNG for extended composition options
- Both needs satisfied with single portrait file

**Quality Standards:**
- Professional lighting and composition
- Clean background
- Subject positioned for proper framing at both crops
- High resolution (minimum 1000px width)
- PNG format for transparency options

**Example Reference:**
- See Azar Imranov portrait (row 102 in tracking spreadsheet)
- Figma frame shows professional headshot appearance
- Actual PNG file extends to waist with usable composition throughout

**Team Access:**
- Designers: Create portraits following this standard
- Videographers: Access full PNG files for extended composition
- CRM Team: Use exported 1000x1000 frame views
