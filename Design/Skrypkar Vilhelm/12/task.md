## Task 5: Document Best Practices for Portrait Generation

### Steps:
1. Review the Favour portrait project and challenges encountered
2. Document the problem (poor reference quality) and solution (video screenshot)
3. Create step-by-step guide for handling poor quality references
4. Document post-processing techniques for overexposed images
5. Compile list of best practices for portrait generation with MidJourney
6. Include examples of successful reference types vs. problematic ones
7. Add tips for working with video editors on reference gathering
8. Create checklist for portrait quality control
9. Share document with design team for feedback
10. Save final version to department documentation folder

### Resources and Links:
- [Portrait Spreadsheet - Row 104](https://docs.google.com/spreadsheets/d/1FAbgx0KGP2qezg6Y_HYbhXx-Aj9OB4j-4LZgW3euono/edit?pli=1&gid=0#gid=0&range=104:104)
- [Thumbnail Spreadsheet - Row 9](https://docs.google.com/spreadsheets/d/1x5dOETWOc2pbqsrZn9DaiGH1aub1BwsTGH4Pr5khSLg/edit?pli=1&gid=0#gid=0&range=9:9)
- MidJourney documentation
- Photo editing tools (Photoshop, Photopea, etc.)

### Instructions:

**Document structure:**

**1. Introduction**
- Purpose of the guide
- When to use these techniques
- Overview of common portrait challenges

**2. Reference Quality Assessment**
Create criteria for evaluating reference images:
- Resolution and clarity
- Lighting conditions
- Angle and composition
- Digital artifacts or filters
- Whether it's a photo of a photo

**3. Problem-Solving Strategies**

When reference is poor quality:
- Option 1: Request better reference from HR or candidate
- Option 2: Extract screenshot from video interview
- Option 3: Find similar reference and adapt
- Option 4: Use multiple reference images combined

**4. Working with Video Screenshots**
Step-by-step process:
1. Request video file or interview recording
2. Find frame with best lighting and angle
3. Use video editing software to export high-quality screenshot
4. Evaluate screenshot quality before proceeding
5. Note timestamp for future reference

**5. MidJourney Portrait Generation**
Best practices:
- Upload reference image to MidJourney
- Use appropriate prompt structure for portraits
- Specify style, lighting, and quality parameters
- Generate multiple variations (4-8)
- Select best result before upscaling
- Use --style parameter for consistency

**6. Post-Processing Techniques**

For overexposed portraits:
- Increase contrast using curves or levels
- Deepen blacks without losing detail
- Adjust shadow detail to prevent flatness
- Use selective color correction if needed
- Test on both light and dark backgrounds

For background removal:
- Use AI background removal tools (remove.bg, etc.)
- Manual refinement in Photoshop if needed
- Check edge quality at high zoom
- Ensure hair and fine details are preserved

**7. Quality Control Checklist**
Before delivery:
- [ ] Image resolution meets requirements
- [ ] Background is cleanly removed
- [ ] Lighting and exposure are balanced
- [ ] Colors are natural and accurate
- [ ] No artifacts or distortions
- [ ] Works well on both light and dark backgrounds
- [ ] Client/stakeholder approval received

**8. Communication Tips**
- Set realistic expectations about reference quality
- Explain turnaround time if reference needs improvement
- Document why original reference was inadequate
- Show before/after if relevant
- Request feedback for continuous improvement

**9. Lessons from Favour Portrait Project**
- Photo-of-photo references should be avoided
- Video screenshots are reliable alternative
- Always check multiple frames when using video
- Budget extra time for post-processing challenging references
- Collaborate with video team for better source material
- Test final portrait on actual use backgrounds

**10. Tools and Resources**
- MidJourney for AI portrait generation
- Remove.bg or Photoshop for background removal
- DaVinci Resolve or similar for video screenshot extraction
- Color correction tools
- Reference image quality checkers
